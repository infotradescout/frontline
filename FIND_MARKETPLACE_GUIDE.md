## **?? TROUBLESHOOTING:**

### **Problem: Can't find Epic Games Launcher**

**Solution:**
```
1. Windows Search (Win key)
2. Type: "Epic"
3. Should show: Epic Games Launcher
4. If not found: Re-download from epicgames.com
```

### **Problem: Marketplace tab missing**

**Solution:**
```
1. Update Epic Games Launcher:
   - Settings (top right gear icon)
   - Check for updates
   
2. Make sure you're on "Unreal Engine" tab (not "Store")

3. If still missing:
   - Restart launcher
   - Sign out and sign back in
```

### **Problem: All assets show as "Paid"**

**Solution:**
```
1. Check "Free for the Month" section first
   (Top of marketplace page)

2. Use search + filter:
   - Search: "free"
   - Filter: Price ? Free

3. Some "Free" packs require:
   - Creating Epic account (free)
   - Accepting terms
   - One-time email verification
```

### **Problem: Downloaded asset not in Unreal**

**Solution:**
```
1. Check you added to correct project (Frontline)

2. In Epic Launcher ? Library:
   - Find the asset
   - Check if it says "Installed"
   - Click "Add to Project" again if needed

3. In Unreal Editor:
   - Restart editor
   - Content Browser ? Refresh
   - Check Content/[PackName]/ folder
```

### **Problem: Download fails**

**Solution:**
```
1. Check internet connection

2. Check disk space (need 1-5 GB per pack)

3. Pause other downloads

4. Try:
   - Epic Launcher ? Settings
   - Downloads ? Throttle Limit
   - Set higher or unlimited

5. Restart Epic Launcher
```
