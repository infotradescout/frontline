## ?? **TECHNICAL IMPLEMENTATION**

### **Phase State Machine:**

```cpp
enum class EIslandPhase : uint8
{
    Warmup,          // Safe, no countdown
    MatchStarted,    // 60s countdown active  
    Destroying,      // Active destruction (30s)
    Destroyed        // Fully removed
};
```

### **Network Replication:**

```cpp
// Replicated properties
UPROPERTY(Replicated) EIslandPhase CurrentPhase;
UPROPERTY(Replicated) float DestructionCountdown;
UPROPERTY(Replicated) float DestructionProgress;

// Multicast RPCs
void MulticastPlayWarning(int32 SecondsRemaining);
void MulticastStartDestruction();
```

### **Damage Scaling Formula:**

```cpp
float DamageMultiplier = 1.0f + (DestructionProgress * 4.0f);
float FinalDamage = BaseDamage * DamageMultiplier * DeltaTime;

Examples:
- 0% destruction ? 1x damage (10 HP/sec)
- 25% destruction ? 2x damage (20 HP/sec)
- 50% destruction ? 3x damage (30 HP/sec)
- 75% destruction ? 4x damage (40 HP/sec)
- 100% destruction ? 5x damage (50 HP/sec)
```
