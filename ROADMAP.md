# Solo-Dev Roadmap

## Phase 1: Greybox

KPI: playable test arena.

Build:
- Player movement
- Gun firing
- Health/damage
- Test enemy
- Basic terrain blockout

## Phase 2: Generated Match

KPI: every restart changes the map.

Build:
- Seed system
- Randomized cover
- Randomized loot
- Randomized spawns
- Safe zone/storm

## Phase 3: Combat Loop

KPI: 5-minute match is playable.

Build:
- Bots
- Loot pickup
- Ammo
- Damage zone
- Win/loss screen

## Phase 4: Visual Identity

KPI: screenshots look like a real game.

Build:
- Stylized environment
- UI pass
- Weapon polish
- Sound
- Lighting

## Phase 5: Multiplayer Test

KPI: 2-4 players can join a generated match.

Rule:
- Start this only after the solo prototype works.
