# CUT LIST

If a task does not directly improve the first playable loop, cut it.

## Cut Until After Prototype 0.1

- Full multiplayer backend and services
- Ranked progression systems
- Battle pass and monetization systems
- Cosmetic pipeline
- Account/login services
- Vehicles
- Crafting
- Base building
- Story mode and narrative systems
- Companion apps, dashboards, web tooling
- Graphics-first visual polish before low-end performance lock

## Scope Guard Questions

Before implementing anything, answer all three:

1. Does it improve generate -> loot -> fight -> survive -> win?
2. Does it unblock a current phase KPI?
3. Can it be completed in less than 2 days solo?

If any answer is No, defer it.
