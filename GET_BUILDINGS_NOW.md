## **? TROUBLESHOOTING:**

### **Problem: Can't install Quixel Bridge**

**Solution:**
```
1. Update Epic Games Launcher:
   - Settings ? Check for Updates

2. Check system requirements:
   - Windows 10+
   - 4GB RAM minimum
   - 1GB free space

3. Try manual download:
   - quixel.com
   - Download Bridge standalone
   - Install directly
```

### **Problem: Bridge won't open in Unreal**

**Solution:**
```
1. Launch Bridge from Epic Games Launcher first
2. Make sure it's running
3. Then try Window ? Quixel Bridge in Unreal
4. If still doesn't work:
   - Download assets in Bridge
   - Export to Frontline project folder
   - They'll show up in Content Browser
```

### **Problem: Downloads fail**

**Solution:**
```
1. Check internet connection
2. Check firewall settings
3. Try downloading one at a time
4. Use lower quality (Medium instead of High)
5. Check free disk space (need ~1 GB)
```

### **Problem: Assets don't import**

**Solution:**
```
1. Make sure Unreal Editor is open
2. Make sure project is Frontline
3. Check Content Browser ? Megascans folder
4. Try manual import:
   - Right-click in Content Browser
   - Import to /Game/Buildings/
   - Browse to downloaded files
```
